Classify Chest X-Rays

Use images of chest X-rays to detect signs of pneumonia.

The data uses X-rays which depict clear lungs without any areas of abnormal opacification in the image, compared with X-rays with bacterical and viral pneumonia. Because of this the machine learning algorithm may struggle with images which show no pneumonia but do show other chest issues.

This neural network was created by Andrew Willacy, using the fast.ai library.

Sensitivity: 0.88 // Specificity: 1.0 // Precision: 1.0
